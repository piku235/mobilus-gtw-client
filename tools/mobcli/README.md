```
Command-line client for interacting with the Mobilus Cosmo GTW

Usage:
  mobcli <subcommand> [OPTION...]

      --help  prints help

Commands:
  sub               subscribes to events
  network-settings  makes network settings query and prints the response
  device-settings   makes device settings query and prints the response
  device-list       makes device list query and prints the response
  current-state     makes current state query and prints the response
  session           prints the current session information
  call-events       sends call events message to provided device
```
